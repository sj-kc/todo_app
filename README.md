
# Todo app

The todo app its an application using ES6 Javascript syntax and localStorage.

### Clone the repo

```bash
git clone https://github.com/sungkimcastro/todo_app.git
```

### Install dependencies

```bash
yarn install
```

### Start a local server to preview the site

```bash
yarn dev-server
```
